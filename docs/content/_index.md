## :beer: Homebrew Recipes

To install the product shown in the following table,
at first, open the Terminal window in your macOS.
Then, type `brew tap tamada/tap`.
Finally, type `brew install <PRODUCT_NAME>`.

Or, type `brew tap tamada/tap/<PRODUCT_NAME>`.
